# Eryndor Active Condition Generator

A Vite + React webpage for the creative worldbuilding generator assignment.

## Local preview
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```
The production output will be created in the `dist` folder.

## Vercel settings
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
